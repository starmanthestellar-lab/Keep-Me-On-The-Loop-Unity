# Keep-Me-On-The-Loop-Unity

<h1>Guia para la instalación de Unity </h1>

<br>
<p align="center">
Guía para la instalación de Unity, Visual Code Studio, configuración de Github
Por: Alejandro Alonso
</p> <br>
<h1>Pasos a seguir</h1>


<h2><strong>1. Descargar y configurar Unity Hub.</strong></h2>

<p>   Descargue Unity Hub desde la pagina oficial:
&nbsp;https://unity.com/download 

1.1. Elija la versión acorde a su sistema operativo.

1.2. Ejecute el instalador.

1.3. Abre <strong>Unity Hub</strong> y registrate/inicia sesión en la pagina de Unity.</p> <br>

<br>

<h2><strong>2. Descargar y configurar Unity Editor.</strong></h2>

2.1. En Unity Hub, entra en la sección de Installs (Instalados), y despues en Install Editor (Instalar editor).

2.2. Seleccione una versión del editor.

Version recomendada: 
Unity 2022.3 LTS

2.3. Durante la instalación, elige los módulos deseados, que permiten la exportación de proyectos o el uso de programas varios durante el desarrollo.

Módulos recomendados:
- Microsoft Visual Studio Community
- Windows Build Support
- WebGL Build Support
  <br>

<br>

<h2><strong>3. Descargar Microsoft Visual Studio Community.</strong></h2>

<strong>En caso de no estar el módulo de Visual Studio</strong>, realize lo siguiente:

3.1. Acceda a la pagina oficial:
&nbsp;  https://visualstudio.microsoft.com/es/vs/community/

3.2. Descarga <strong>Visual Studio Community</strong>.

3.3. Durante la instalación, selecciona:

- Desarrollo de juegos con Unity.
